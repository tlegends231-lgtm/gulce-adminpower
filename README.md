***Minecraft Datapack Project (GulceOS)**
Bu depo, **Minecraft 1.21.6+** için geliştirilmiş **vanilla datapack** içerir.  
**Mod kullanmaya gerek yoktur**; oyun içi mekanikler, özel sistemler ve araçlar için tasarlanmıştır.

# **Özellikler**
+ %100 Vanilla uyumlu
+ Özel komut ve işlevler
+ GUI ve dialog sistemleri
+ Permission ve makro sistemi entegrasyonu
+ `/function custom_admin:stop-` ile "Devre dışı" bırakma ve Daha ***Fazlası***
